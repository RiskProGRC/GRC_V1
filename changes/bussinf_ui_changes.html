6. Global Table Styling Standard — Reusable Pattern

Saved CSS Moved to assets/css/style.css — Available Everywhere

The .table-buss class was extracted from page-level <style> blocks and added to the project's shared stylesheet so it is available on every page without copying CSS. Applied so far: bussinf.php, inbox.php.

assets/css/style.css — appended at end of file
/* ── GRC standard table style ── */
.table-buss { border-collapse: collapse; }
.table-buss th {
    font-size: 12px; font-weight: 700; color: #fff;
    background: #02338d; padding: 3px 5px;
    white-space: nowrap; text-align: center; vertical-align: middle;
    border: 1px solid rgba(255,255,255,0.3);
}
.table-buss td {
    font-size: 12px; font-weight: 500; color: #222;
    padding: 2px 5px; text-align: center; vertical-align: middle;
    white-space: nowrap; border: 1px solid #b8c8de;
}
.table-buss tbody tr:hover td { background: #eef4ff; }

Template HTML — Apply to Any Table

Wrap the table in table-responsive and add table-buss to the class list. Replace tableN with the actual ID.

Copy this pattern to any page
<div class="table-responsive">
    <table class="table table-striped table-buss" id="tableN">
        <thead>
            <tr>
                <th>Column 1</th>
                <th>Column 2</th>
            </tr>
        </thead>
        <tbody>
            <!-- rows -->
        </tbody>
    </table>
</div><!-- table-responsive -->

Template JS — DataTable Init with Null Guard

Always use the null guard form. If the table doesn't exist on a given page load (e.g. hidden in a tab), it won't throw an error.

Without guard — throws if element missing
let tableN = document.querySelector('#tableN');
let dataTableN = new simpleDatatables.DataTable(tableN);
With guard — safe on every page
let tableN = document.querySelector('#tableN');
if (tableN) new simpleDatatables.DataTable(tableN);
