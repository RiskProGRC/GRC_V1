What Was the Problem
Cleanup

~430 Lines of Dead Code from Mazer Template

layout/nav.php was 627 lines long, but only ~197 lines were active working navigation. The remaining ~430 lines were commented-out HTML blocks left over from the Mazer UI template that was used as the base theme — none of it was ever used or intended to be used.

Blocks removed:

  • 3-level submenu example inside Inbox (Widgets, Icons, Charts with sub-items)
  • Component examples in Risk Tracker submenu (Breadcrumb, Button, Card, Carousel, Dropdown, Modal, Navs, Pagination, Spinner, Tooltip, Avatar, Sweet Alert, Toastify)
  • Form editor submenu inside Risks (Quill, CKEditor, Summernote, TinyMCE)
  • Commented-out KRI menu item in Risk Monitor
  • Commented-out Owner & Nominee items with Google Map / JS Vector Map sub-items
  • Commented-out Photo Gallery & Checkout Page items
  • Full Support menu (Documentation, Contribute, Donate links)
  • Inline "3 Level Submenu" text labels scattered through the markup
