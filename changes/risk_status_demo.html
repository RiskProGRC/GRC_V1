Overview

The first rebuild used a JSON API pattern — PHP returned structured JSON, JavaScript built every table with string builders. It worked but was complex: 5 builder functions, a dtMap tracker, fetch(), ob_start(), and 250+ lines of JS.

This second version matches how the rest of the project works: the fetch file echoes raw HTML (dept info + Bootstrap tabs + 5 tables) exactly like entityAction.php, and the main page uses a single $.ajax POST to inject it — exactly like the global display() function in app.js.

loadDept() — the entire JS (12 lines)

JS
function loadDept(id) {
    if (!id) {                                          // blank option selected — reset
        $('#rv-display').html('...prompt...');
        return;
    }
    $('#rv-display').html('Loading...');             // feedback while request is in flight

    $.ajax({ type: 'POST', url: 'riskview_fetch.php', data: { deptid: id },
        success: function(html) {
            $('#rv-display').html(html);              // inject the full HTML response
            ['rv-tab1','rv-tab2','rv-tab3','rv-tab4','rv-tab5'].forEach(function(tid) {
                var el = document.getElementById(tid);
                if (el) new simpleDatatables.DataTable(el, { perPage: 10 }); // null-guard
            });
        },
        error: function() { $('#rv-display').html('Failed to load...'); }
    });
}

Same pattern as the project's existing display() function in assets/js/app.js:1113.

require_once 'AuthGuard.php';      // session check

if (!$deptid) exit;

// 6 mysqli_query() calls with verified column names
$dept_q = mysqli_query($con, "SELECT ... FROM department ...");
$risk_q = mysqli_query($con, "SELECT ... FROM risk ...");
$ki_q   = mysqli_query($con, "SELECT ... FROM ki ...");
$ctrl_q = mysqli_query($con, "SELECT ... FROM \`control\` ...");
$act_q  = mysqli_query($con, "SELECT ... FROM action ...");
$rmd_q  = mysqli_query($con, "SELECT ... FROM recommend ...");

$dept = mysqli_fetch_assoc($dept_q);
if (!$dept) exit;
// then: echo HTML — dept fields, nav-tabs, 5 while() table loops
while ($r = mysqli_fetch_assoc($risk_q)) { echo '<tr>...</tr>'; }
